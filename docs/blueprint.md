# **App Name**: FlavorFusion Events

## Core Features:

- Service Showcase: Display a comprehensive catalog of catering services with detailed descriptions and pricing.
- Event Gallery: Showcase high-quality images of past catering events and setups.
- Booking System: Enable clients to submit detailed catering service booking requests through an intuitive form, stored in Cloud SQL.
- Inquiry Contact Form: Allow users to send inquiries and custom requests to the catering company, stored in Cloud SQL.
- AI Menu Suggestion Tool: Provide personalized menu recommendations based on event type, dietary preferences, and budget, utilizing AI reasoning to determine optimal food choices.
- Admin Event Management: An administrative interface for staff to manage and review incoming booking events and contact inquiries stored in Cloud SQL.

## Style Guidelines:

- Primary Color: Muted terracotta (#B97F70). This warm, inviting hue creates a sophisticated yet welcoming ambiance, reflecting culinary delight and professional service.
- Background Color: Soft ivory-peach (#F7F0EE). A very lightly desaturated version of the primary hue, providing a clean, airy canvas that enhances readability in a light color scheme.
- Accent Color: Elegant rose-mauve (#D16C8F). An analogous shade to the primary, chosen for its vibrancy and contrast, ideal for drawing attention to calls-to-action and key elements.
- Headline font: 'Playfair', a modern serif, conveying elegance and a high-end feel. Body font: 'PT Sans', a humanist sans-serif, for clear and modern readability.
- Ensure a clean and structured layout with clear, easy-to-navigate sections for services, event galleries, and booking functionalities.
- Utilize elegant, minimalist icons that subtly represent various catering services and different types of events, enhancing the user interface without clutter.
- Implement subtle transitions and animations, especially on image hover effects within the gallery and smooth page transitions, to create a sophisticated and engaging user experience.
